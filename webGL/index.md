# 3、GPU、CPU

CPU更适合做计算复杂的操作，GPU适合做计算简单但是计算量大的操作。无论有多少个像素点，都可以在着色器程序中被GPU同时执行。

# 4、webGL绘制步骤

1. 创建webGL上下文
2. 创建webGL程序
3. 将数据存到缓冲区
4. 将缓冲区数据读取到GPU
5. GPU执行webGL程序，输出结果
